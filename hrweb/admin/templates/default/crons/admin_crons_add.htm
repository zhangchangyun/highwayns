ヒント：

计划任务是一项使系统在规定时间自动执行某些特定任务的功能，在需要的情况下，您也可以方便的将其用于功能的扩展。
计划任务是与系统核心紧密关联的功能特性，不当的设置可能造成功能的隐患，严重时可能导致网站无法正常运行。

タスク計画を追加します
{#$inputtoken#}
名前：
このタスクのタスク名を設定します
タスクのスクリプト：
このタスクの実行ファイル名の設定は、パスが含まれないよう、./include/crons/目録中
毎週：
週の設定日は、この設定は、以下の「日」を設定を上書きし、制限されていない「*」、このタスクを実行するには
毎月：
任務実行日の設定、「*」が制限なし
時間：
このタスクを実行する時間が設けられています、「*」は制限なし
分：
このタスクを実行する用意分、空白を制限しません
利用可能：
設置は使えないとタスクが行わない
