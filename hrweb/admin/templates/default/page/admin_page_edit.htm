キャッシュの説明
クエリがデータをキャッシュしている場合は、同じクエリがデータベースを照会する必要はありません後、シリアライズ後の結果をファイルに保存されますが、キャッシュから獲得します。この改善は、スピードを向上させるためのプログラムがあまりにも大きさになります。
缓存时间是缓存重新加载周期，周期越长，数据库的负荷越小，缓存周期假设为50秒，则每50秒刷新缓存一次。
